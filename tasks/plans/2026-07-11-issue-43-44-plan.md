## Design

### #43 — oldest mtime
`getCacheAge` currently returns `Date.now() - newest`. Change to `Date.now() - oldest`. A partial failure that rewrote 7 of 8 `_latest.csv` files leaves the 8th at its old mtime → oldest is old → age exceeds window → `ensureFresh` re-exports. All-8-fresh → oldest is recent → no re-export (existing behavior preserved). The existing "any table file missing → re-export" branch in `ensureFresh` still covers the never-succeeded-at-all case.

### #44 — in-flight dedup
```ts
const inFlightRefreshes = new Map<string, Promise<Record<string, string>>>();

function dedupedExportAll(username, password, cacheDir) {
  const existing = inFlightRefreshes.get(cacheDir);
  if (existing) return existing;
  const p = exportAll(username, password, cacheDir)
    .finally(() => inFlightRefreshes.delete(cacheDir));
  inFlightRefreshes.set(cacheDir, p);
  return p;
}
```
`ensureFresh` calls `dedupedExportAll` in both its `exportAll` sites. Map keyed by `cacheDir` honors per-instance `CT_CACHE_DIR`. Slot cleared on settle (success or failure) via `.finally`; a rejected shared refresh rejects all current waiters, and the next call starts a fresh refresh.
